## üöÄ How to Run

1.  **Clone the repository**
    ```bash
    git clone https://github.com/ClauDevy/anchor-app.git
    cd anchor-app
    ```

2.  **Install Dependencies**
    This downloads the required packages (Express, etc.) listed in `package.json`.
    ```bash
    npm install
    ```

3.  **Start the Server**
    Run the application locally.
    ```bash
    node server.js
    ```

4.  **Open in Browser**
    You will see a message in your terminal: `Anchor Server running at http://localhost:3000`.
    Open your web browser and go to:
    ```
    http://localhost:3000
    ```
